Bagaimana Cara Kerjanya?

Perceptron Algorithm

Jika titik salah klasifikasi, update:
w = w + α·y·x
b = b + α·y
Bergerak menuju titik yang salah untuk memperbaiki batas keputusan.

Sigmoid Function

Mengubah nilai linear menjadi probabilitas:
σ(z) = 1 / (1 + e⁻ᶻ)
dimana z = w₁x₁ + w₂x₂ + b

Log-Loss (Cross-Entropy)

Error function:
L = -[y·ln(ŷ) + (1-y)·ln(1-ŷ)]
Memberi penalti besar untuk prediksi yang jauh dari label.

Gradient Descent

Update weights menggunakan gradien:
w = w - α · ∂L/∂w
∂L/∂wⱼ = (ŷ - y) · xⱼ
